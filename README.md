A. Description: AI-powered system to detect crisis situations from image, text, or audio and provide emergency guidance.

B. Problem: In emergency situations, people often panic and don’t know what actions to take. 
There is a need for a smart system that can quickly analyze a situation and provide guidance.

C. Solution:
Based on the input (audio, photos, voices, maps, sos systems)
1. Detects and analyzes the type of calamity
2. Classifies the severity of the calamity
3. Suggest imidiate actions for maximum Survivability 
4. Calculates the route of nearby hospital 

D. Features:
- Image-based crisis detection
- Voice input (speech-to-text)
- Text input support
- AI reasoning engine
- Severity classification (Low, Medium, High)
- Actionable emergency steps
- Smart action buttons (Call, Alert, Share Location)
- Chaining technique for fast help (using BT or sattlite)
